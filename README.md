# Labyrinth

Labyrinth 用于将视频画面混淆化，从而规避一些版权限制。

!!EXPERIMENTAL VERSION!!

警告：请遵守当地法规，不要用于非法用途。

## 安装

下载源代码：

`git clone https://github.com/ERR0RPR0MPT/Labyrinth.git`

安装依赖：

`pip install -r requirements.txt`
